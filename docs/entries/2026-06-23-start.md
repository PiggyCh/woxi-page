# Start

This repository is a public place for continuous writing.

Write thoughts here as Markdown files. Keep each file focused on one idea, decision, observation, plan, or raw note. The `ai-index.json` file gives AI tools a stable map of the archive, so future tools can find and summarize the content without guessing the structure.

Suggested writing pattern:

- Capture raw thoughts quickly.
- Add tags only when they help retrieval.
- Use summaries for entries that should be easy for AI to scan.
- Do not write secrets, credentials, or private personal data here.
